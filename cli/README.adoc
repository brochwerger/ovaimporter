# Import OVAs into OCP-V from the command line

[source]
--
Usage: ova2bootvolume -u <URL>|-d <directory>

Options:  
 -h         Print this Help.  
 -u <URL>   Download OVA from <URL>  
 -d <directory>  Use OVA in <directory>  
--
